  * Removed deprecated function calls. (Issue #3034195)

    (Issue #3033650)

  * Adds support for Drupal core 8.7.
  * Adds a description to an administrative link. (Issue #3034041)
* Updated Lightning Workflow to 3.5, which:

    list page. (Issue #3034047)
